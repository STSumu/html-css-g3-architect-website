# G3 Architects – Landing Page

This project is a responsive landing page built using **HTML** and **CSS** as part of a guided web development course. It was developed by following instructor-led video tutorials and a provided Figma design.

## Features

* Responsive navigation bar
* Hero/banner section
* Team showcase section
* Feature highlights section
* Statistics/facts section
* Sponsors section
* Clean footer layout

## Technologies Used

* HTML5
* CSS3
* Google Fonts
* Flexbox layout

## Design Reference

This project was built based on a provided Figma design and implemented step-by-step following instructor guidance.
Original Figma Design : https://www.figma.com/proto/sZAFkcbMt35KjS9hp4gKfH/g3-architects?node-id=2-2

## Learning Context

This was a guided practice project where I learned:

* How to structure multi-section landing pages
* How to convert Figma designs into code
* How to use Flexbox for layout design
* How to organize CSS for larger page structures
* How professional landing pages are built step-by-step
